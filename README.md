# plugin

**Optimized Lampa plugin** — плагін для перегляду фільмів/серіалів через Lampa (оптимізована версія).

## Основні зміни
- Кнопка у повному перегляді: **«Дивитись ТУТ»** (українська локалізація).
- Всі відомі балансери активовані за замовчуванням.
- Поліпшена стабільність та продуктивність.
- Готовий single-file: `plugin.js` — можна завантажити в корінь репозиторію.

## Як встановити плагін у Lampa
1. Переконайся, що репозиторій публічний.
2. Використай **raw**-URL або CDN для підключення у додатку:
   - Raw: `https://raw.githubusercontent.com/DesRums/bwarc-plugin/main/plugin.js`
   - Рекомендуємо (більш стабільно для CORS): `https://cdn.jsdelivr.net/gh/DesRums/bwarc-plugin@main/plugin.js`
3. У налаштуваннях Lampa → Plugins / External → додай URL плагіна і перезавантаж плагіни/додаток.
4. Якщо зелена помилка (не вдалося завантажити) — дивись розділ «Troubleshooting».

## Troubleshooting
- **Файл порожній?** — відкрий `plugin.js` у GitHub і переконайся, що там код (не «This file is empty»).
- **403/404 або CORS** — спробуй CDN jsDelivr (посилання вище).
- **Кеш у Lampa** — повністю перезапусти додаток, очисти кеш плагінів.
- **Якщо помилка RCH / nws** — інтернет/блокування хостів може вплинути; спробуй інше підключення або CDN.
